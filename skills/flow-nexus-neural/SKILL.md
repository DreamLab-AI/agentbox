# Flow Nexus Neural

> ⚠️ **Not installed in this environment.** Requires Flow Nexus account and cloud sandbox access.

## What it does

Trains neural networks in isolated E2B sandboxes:
- Feedforward, LSTM, GAN, transformer architectures
- Distributed training across cloud sandbox fleet
- No local GPU required

## To use this skill

Requires Flow Nexus credentials configured.

## Alternative (local training)

For local GPU-based training, use `pytorch-ml` or `cuda` instead.
