# Flow Nexus Swarm

> ⚠️ **Not installed in this environment.** This skill requires a Flow Nexus account and cloud access.

## What it does

Deploys and coordinates AI swarms on Flow Nexus cloud infrastructure:
- Event-driven workflow automation
- Message queue processing with intelligent routing
- Scale-out agent coordination beyond local resource limits
- Persistent cloud-hosted swarm state

## To use this skill

1. Create a Flow Nexus account
2. Configure your API credentials
3. The skill will become available once credentials are set

## Alternative (local swarms)

For local multi-agent orchestration without cloud dependency, use `swarm-advanced` instead.
